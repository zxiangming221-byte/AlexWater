# AlexWater

Android 喝水提醒应用。定时提醒 + 记录追踪 + 统计回顾。

## 功能

- 定时喝水提醒（每小时 / 1.5小时 / 2小时 / 自定义）
- 日历集成 + AlarmManager 双保险提醒
- 饮水记录与快捷按钮
- 周/月统计图表
- 暗色 / 浅色主题
- 免打扰时段

## 技术栈

Kotlin + Jetpack Compose + Material 3

## 运行

用 Android Studio 打开，Sync Gradle，Run。

最低 Android 8.0 (API 26)，目标 Android 14 (API 34)。
