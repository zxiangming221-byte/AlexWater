核心特性

提醒 + 记录 + 统计

三个维度覆盖日常饮水管理

5 种提醒方案

每小时、1.5 小时、2 小时、自定义间隔、自定义时间点。想怎么设怎么设。

日历集成提醒主力

直接写入系统日历事件，系统级投递。手机重启照样响，通知方式在日历 App 自己调。

AlarmManager 双保险

setAlarmClock() 最高优先级 API 做后备。双重保障，不响的概率降到最低。

周/月统计 + 趋势

柱状图可视化每日摄入、7 日均值趋势、时段分布分析。数据说话，不靠感觉。

进度环 + 实时反馈

Canvas 手绘进度环，spring 弹性动画。达标后脉冲庆祝，仪式感拉满。

暗色 + 浅色双主题

Material 3 全 token 化配色。逐色 250ms 过渡动画，切换不闪烁。
